SWMM - basic

Extract SWMM results for a collection system to a pandas DataFrame.

Overview

Tip

MIKE IO 1D treats SWMM and Res1D results in the same way, so you may also refer to Res1D examples.

import mikeio1d
res = mikeio1d.open("../data/swmm.out")
