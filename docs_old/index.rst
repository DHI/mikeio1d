MIKE IO 1D: Read MIKE 1D in python
==================================

Facilitate reading res1d and xns11 data files.

For other MIKE files (Dfs0, Dfs1, Dfs2, Dfsu,...) use the related package [MIKE IO](https://github.com/DHI/mikeio)

This package is simple to install and deploy operationally and is accessible to everyone as open-source.
